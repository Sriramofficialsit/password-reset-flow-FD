## Password Reset Flow


This repository demonstrates a complete Password Reset Flow implemented using Node.js for the backend and React for the frontend. The flow includes sending password reset links via email and updating the password securely.

## Features
User-friendly password reset mechanism.

Backend functionality to generate password reset tokens.

Email integration for sending password reset links.

Frontend implementation for resetting the password.

Secure handling of tokens and user data.

## Tech Stack

## Backend:

Node.js

Express.js

Nodemailer (for email functionality)

MongoDB with Mongoose (for database management)

dotenv (for environment variables)

crypto (for token generation)

## Frontend:

React.js

Tailwind CSS (for styling)

Axios (for API requests)

## Installation

Ensure you have the following installed:

Node.js

MongoDB (local or cloud-based)

npm or yarn
